# Classificadores — Machine Learning

Atividade prática com 3 sistemas de classificação independentes, cada um treinado em um dataset real e com pipeline completo de pré-processamento, treinamento e inferência.

## Questões

| Pasta | Dataset | Tarefa |
|---|---|---|
| [`questao1-heart-failure/`](./questao1-heart-failure) | Heart Failure Clinical Records | Prever sobrevivência ou risco de óbito |
| [`questao2-wine-quality/`](./questao2-wine-quality) | Wine Quality | Classificar qualidade do vinho (3 a 9) |
| [`questao3-black-friday/`](./questao3-black-friday) | Retail Black Friday Sales | Prever categoria, pagamento e faixa etária |

Cada pasta tem seu próprio README com detalhes do pipeline, modelo utilizado e saídas geradas.
